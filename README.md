## Frontend

To run the frontend, navigate to the `swiftshop` directory and use the following command:

```terminal
cd swiftshop/swiftshop/src/app
ng serve
```

### Backend
To run the backend API, navigate to the API directory and use the following command:

```terminal
cd swiftshop/swiftshop/src/app
python API.py
```
